# HZXOnlinePlayer
> 这是一个基于[Meting](https://github.com/metowolf/Meting)的在线音乐播放器，具有歌曲播放、下载、搜索、歌词同步滚动显示等功能，其中歌曲搜索支持网易云、QQ、虾米、酷狗、百度音乐源切换。

在线预览[http://hzx2034252646.coding.me/music](http://hzx2034252646.coding.me/music)
